libgadu 1.12.1

30th December 2014

Changes

  • Better compatibility with Gadu-Gadu 11 protocol.
  • Fixed compilation on some operating systems.
  • protobuf-c library updated to 1.0.2.
  • Fixed memory violation when incorrect contact search server response is received.
  • Fixed off-by-one write on stack when too long proxy server repoonse is received.
  • Fixed memory corruption when incorrect file transfer or voice call proxy server response is received.

More details description of API/ABI changes is available in documentation (Polish only).

Download

  • libgadu-1.12.1.tar.gz

Known issues

  • Some features of previous protocol versions (file transfers, voice calls) may not be supported by the network or be unstable.
  • Not all protocol features are unit tested.
