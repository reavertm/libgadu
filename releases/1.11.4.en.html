libgadu 1.11.4

8th May 2014

Changes

  • Security fix in file transfer via proxy server. A crafted server response may cause memory corruption (CVE-2014-3775)

Download

  • libgadu-1.11.4.tar.gz

Known issues

  • Encrypted connection doesn't verify server certificate which may lead to man-in-the-middle attack. This is a potential security issue fixed in version 1.12.0.
  • Voice calls and transfers via proxy server are not well tested.
