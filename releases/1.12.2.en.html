libgadu 1.12.2

1st February 2017

Changes

  • Fixed memory corruption when updating chat participant list.
  • Fixed error handling with GnuTLS.
  • Updated TLS connection address.
  • Fixed disconnection issue when joining a chat room with a title set.
  • Use of more secure random functions.
  • Fixed hanging when sending files under Windows.
  • Fixed out-of-memory behavior.
  • Resolver fixes for Windows.
  • protobuf-c library updated to 1.2.1.
  • Better Windows build process.
  • Fixed unit tests.

More details description of API/ABI changes is available in documentation (Polish only).

Download

  • libgadu-1.12.2.tar.gz

Known issues

  • Some features of previous protocol versions (file transfers, voice calls) may not be supported by the network or be unstable.
  • Not all protocol features are unit tested.
