libgadu 1.12.0

13th June 2014

Changes

  • Gadu-Gadu 11 protocol support.
  • An option to verify server certificate.
  • An option to use own functions for TCP/TLS connections.
  • An option to pass server domain name, not just IP address.
  • Fixed small issues reported by Pigdin project found by their Coverity audit.
  • OpenSSL is not detected automatically.
  • Picture transfer fixes.
  • Fixed unit tests reliability.
  • Fixed Windows port reliability.

Because lack of server certificate verification was reported as a security issue please treat this release as critical if encrypted transmission was in fact used.

More details description of API/ABI changes is available in documentation (Polish only).

Download

  • libgadu-1.12.0.tar.gz

Known issues

  • Resolver unit tests may not work properly in Fedora.
  • Some features of previous protocol versions (file transfers, voice calls) may not be supported by the network or be unstable.
  • Not all protocol features are unit tested.
